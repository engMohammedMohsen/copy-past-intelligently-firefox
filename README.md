A powerful Firefox extension for managing tabs and clipboard URLs with intelligent copy and paste features.

## Installation

1. Open Firefox
2. Go to `about:addons` or press `Ctrl+Shift+A`
3. Click on "Extensions" in the left sidebar
4. Click the gear icon and select "Install Add-on From File"
5. Navigate to the extension folder and select the manifest.json file
6. Click "Install" to complete the installation

## Features

- **Copy All Tabs** - Quickly copy URLs from all open tabs to your clipboard
- **Smart Paste** - Intelligently paste URLs with multiple paste modes:
  - Regular paste
  - Reverse paste (open tabs in reverse order)
  - Random single URL (open a random URL from clipboard)
  - Random all URLs (open all URLs from clipboard in random order)
- **Format Copy** - Copy URLs with custom formatting options
- **Smart Paste Detection** - Automatically identify and handle URLs in clipboard

## How to Use

### Quick Access

The extension adds a toolbar button (icon) to your Firefox toolbar. Click it to access the main menu.

### Copy All Tabs

1. Click the extension icon in the toolbar
2. Click the **نسخ** (Copy) button
3. All URLs from your currently open tabs will be copied to your clipboard

**Keyboard Shortcut:**

- Windows/Linux: `Alt+C`

### Paste URLs

Click the extension icon and use one of these paste options:

- **لصق** (Paste) - Open all URLs from clipboard in new tabs
- **لصق المواقع بالعكس** (Paste in Reverse) - Open URLs in reverse order
- **لصق موقع عشوائي** (Paste Random Single) - Open one random URL from clipboard
- **لصق المواقع عشوائياً** (Paste All Randomly) - Open all URLs in random order

### Settings

Access advanced options by:

1. Clicking the extension icon
2. Clicking the settings link at the bottom of the popup

Available settings:

- **Smart Paste** - Enable intelligent URL detection and handling
- **Format Copy** - Customize how URLs are formatted when copied

## Tips & Tricks

1. **Copy Multiple Tab Groups** - Open tabs in different windows and copy all windows separately
2. **Use with Text Editors** - Paste copied URLs into any text editor that supports markdown lists
3. **Create URL Collections** - Copy and save URL collections to your clipboard, then paste them back later
4. **Combine with Smart Paste** - Enable smart paste in settings for automatic URL detection and formatting

## Troubleshooting

**Extension doesn't appear in toolbar:**

- Go to `about:addons` and enable the extension if it's disabled

**Paste not working:**

- Make sure the clipboard contains valid URLs
- Check that "Smart Paste" is enabled in settings if needed

**Can't copy all tabs:**

- Ensure the extension has permission to access tabs (check in browser permissions)
